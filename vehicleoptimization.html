Vehicle Route Optimization Using Fulfillment Center Simulation

Optimization Theory · REST API · Python

With autonomous robots available to service 3 intake locations and 3 docks. The goal is to take as many packages as possible from the intake locations and move them to the docking locations.

Optimization Theory

For this project, we are going to take a look at a single robot in the warehouse, and optimize it's path such that the delivered packages are maximized. Each intake or dock has a number of packages that can be picked up or dropped off, respectively. Additionally, each location has a distance from the robot. We define the trip's efficiency as the number of packages divided by the distance. Then, we calculate this number for each possible trip and choose the highest. For further optimization reading, see the complete paper.

Code Base

The project communicates with a web application using a REST API. Once the development environment was set up, a script was made to GET warehouse data, preform optimization, and POST robot commands. Source code is available on Github.
